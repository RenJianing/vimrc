## INSTALL

Get vim plug to manage the plugins:

```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

open a doc wit vim and run:

```
:PlugInstall
```

## USEAGE

Search file in vim:
'\'+'f'

Search symbol in vim:
'Ctrl+F' and 'Enter'

Select target with:
'Ctrl+j' and 'Ctrl+k'

and jump to with:
'Enter'

Jump back with:
'Ctrl+o'
